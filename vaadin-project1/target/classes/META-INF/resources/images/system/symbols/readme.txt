Diese Folder ist für alle System Icons (Untergruppe Symbols), die in System verwendet wird, angelegt.

Diese beeinhaltet Symbole, die in allen Applikationen verwendet werden können.

Für weitere Symbole kann auch das Text-Command für das GliIcon eingesetzt werden 
(Text als HTML entity -> https://dev.w3.org/html5/html-author/charref):
Bsp. für ein Image(string) mit einem "Bullet": "{text, 200, red}&bull;"
